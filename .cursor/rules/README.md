# Cursor Rules для GradeApp

В папке `.cursor/rules/` можно размещать правила и подсказки для Cursor (файлы .mdc или .md), которые автоматически подхватываются при работе с проектом.

## Рекомендуемое использование

1. **Ссылаться на документацию** — в правилах можно указывать: «При изменении API сверяйся с .cursor/api.md и .cursor/db.md».
2. **Соглашения по коду** — кратко дублировать ключевые пункты из .cursor/code-style.md (Black, Flake8, ESLint, алиасы, i18n).
3. **Контекст проекта** — напоминание: GradeApp — образовательная платформа для школ Казахстана, монолит Django + React, роли привязаны к школе.

## Текущая документация

Вся основная документация лежит в `.cursor/` (не в rules):

- **instructions.md** — главные инструкции для Cursor
- **CONTEXT.md** — краткая справка
- **architecture.md**, **api.md**, **db.md**, **frontend.md**, **backend.md**, **code-style.md**, **business-logic.md**, **branding.md**, **requirements.md**
- **glossary.md** — глоссарий терминов

При создании собственных правил в `.cursor/rules/` используй эти файлы как источник правды и ссылайся на них по необходимости.
